# International-Education-Cost-Analysis
A Python-based analysis of global tuition and living costs using Pandas, Matplotlib and Seaborn
International Education Cost Analysis

## Overview
This project explores the cost of higher education across different countries, focusing on tuition fees and living expenses for international students.

## Tech Stack
- **Python / Pandas:** For data cleaning and transformation.
- **Matplotlib / Seaborn:** For visualizing cost distributions.

## 📉 Analysis Goals
- Identify the most expensive vs. most affordable countries for students.
- Analyze the correlation between tuition and living costs.
- Determine regional cost trends.
